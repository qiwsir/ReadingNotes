# 老齐的读书笔记

## 网站：http://itdiffer.com

## 已经出版的书籍

- 《跟老齐学Python：轻松入门》

  面向初学者、零基础读者。以Python3为教学语言。全面详实、深入浅出地讲解Python语言。

- 《跟老齐学Python：Django实战》

  针对Django 1.x版本。突出实战性。

- 《跟老齐学Python：Django实战（第二版）》

  针对Django 2.x版本。

- 《跟老齐学Python：数据分析》

  数据科学、机器学习必读，介绍NumPy、Pandas、Matplotlib等最基础工具在数据分析领域的应用。

- 《Python大学实用教程》

  适合于大学生学习
